Nova programming language

Goals

The goal of the language is to be as human-readable as possible, while maintaining powerful capabilities.

Open source

The language is licensed under GNU GPL v3.0, meaning it will stay open-source forever - and everything that utilizes it must also be open-source (licensed under GNU GPL v3.0, to be exact)

Active development

I actively work on the language, expanding its capabilities and upgrading current ones.

Community-driven

I want to work on Nova with the community. You're welcome to fork the repo, make pull requests, suggest new features, etc.
