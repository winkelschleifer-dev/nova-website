Nova Programming Language

A simple, powerful and human-readable programming language.

Goals

Nova aims to be as human-readable as possible, while maintaining powerful capabilities and a clear structure.

Open Source

Nova is licensed under GNU GPL v3.0. This ensures that the language implementation stays open-source and that improvements can be shared with the community.

Active Development

Nova is actively developed and improved over time, with new features and improvements being added regularly.

Community Contributions

Everyone is welcome to contribute to Nova. You can report issues, suggest features, create pull requests, or build tools around the language.
